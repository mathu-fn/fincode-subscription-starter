# Contributor Covenant Code of Conduct

## 私たちの誓い

オープンで歓迎的なコミュニティを育てるため、コントリビューター・メンテナーとして、年齢、体型、目に見える/見えない障害、民族性、性別の特性、ジェンダー・アイデンティティと表現、経験のレベル、教育、社会経済的地位、国籍、容姿、人種、宗教、性的指向や性的アイデンティティにかかわらず、誰にとっても嫌がらせのないプロジェクト参加体験を保証することを誓います。

## 私たちの基準

ポジティブな環境作りに貢献する行動の例:

- 思いやりと共感を示す
- 異なる意見・視点・経験を尊重する
- 建設的なフィードバックを与え、また受け入れる
- 自分の過ちに責任を持ち、影響を受けた人に謝罪し、学ぶ
- 個人ではなく、コミュニティ全体にとって最善のことに集中する

許容されない行動の例:

- 性的表現や性的注意・誘いかけ
- トローリング、侮辱、軽蔑的なコメント、個人攻撃や政治的攻撃
- 公的・私的な嫌がらせ
- 明示的な許可なく、住所やメールアドレスなどの他者の個人情報を公開すること
- 業務上不適切と合理的に判断されるその他の行為

## 適用範囲

本行動規範は、プロジェクトのすべての場 — Issue、Pull Request、Discussions、コミットメッセージ、関連するチャットやイベント — で適用されます。また、コミュニティを公的に代表している場面（公式メールアドレスの使用、公式 SNS アカウントでの投稿、オンライン/オフラインイベントでの代表行為など）にも適用されます。

## 違反の報告

虐待的、嫌がらせ、その他許容されない行動を見つけた場合は **非公開で** メンテナーへ報告してください。本リポジトリでは **GitHub の Private Vulnerability Reporting** を共通の非公開窓口として運用しています。手順は [SECURITY.md](./SECURITY.md#脆弱性の報告方法) を参照してください。

報告は機密として扱われます。報告者・対象者の特定情報は、適切な対応に必要な範囲を超えて共有されません。

## 執行

メンテナーは行動規範に違反する行動を判断し、適切と判断される対応を取る責任と権限を持ちます。対応には、コメントの編集・削除、Issue / PR のクローズ、コントリビューターの一時的または永続的な参加禁止が含まれます。

## 帰属

本行動規範は [Contributor Covenant](https://www.contributor-covenant.org) version 2.1 を基にしています。原文: <https://www.contributor-covenant.org/version/2/1/code_of_conduct.html>
