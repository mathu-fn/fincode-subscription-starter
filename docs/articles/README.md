# Launch articles

公開用の技術記事の下書きを置くディレクトリです。

これらはリポジトリの一部としてのドキュメントではなく、**Zenn / Qiita / dev.to などの外部プラットフォームへ投稿するための下書き** です。プロジェクトの認知を広げ、フィードバックを集めることを目的としています。

## 記事一覧

| ファイル | 概要 | 想定プラットフォーム |
| --- | --- | --- |
| [2026-05-introducing-fincode-subscription-starter.md](./2026-05-introducing-fincode-subscription-starter.md) | プロジェクトの紹介記事。Fincode 連携リファレンスの不足という課題、機能、アーキテクチャ、設計のポイント。 | Zenn（フロントマター付き） |

## 使い方

1. 投稿したい記事を選び、内容を確認・必要に応じて加筆する。
2. フロントマターを投稿先プラットフォームの形式に合わせる（Zenn 形式で書いてあります）。
3. `published: false` を `true` に変えて投稿する。
4. 投稿後、記事 URL をこの表に追記しておくと、後から参照しやすくなります。

> 下書きの段階では `published: false` のままにしてあります。
